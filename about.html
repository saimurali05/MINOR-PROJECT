About MiniWallet

MiniWallet is a demonstration project built to showcase modern web technologies for interacting with the Ethereum blockchain. It is a fully client-side application designed for educational purposes.

Technology Stack

This application is built with a focus on simplicity and performance, using:

  • HTML5 & CSS3: For structure and modern, responsive styling.
  • JavaScript (ES Modules): For all application logic, ensuring a clean and modular codebase.
  • Viem: A lightweight TypeScript interface for Ethereum, used for blockchain interactions.
  • Etherscan API: For fetching transaction history data.
  • Node.js/Express: For OTP-based email verification.

Security & Privacy

Your security is paramount. MiniWallet operates entirely within your browser. This means:

  • No Server Storage: Your private keys are never sent to or stored on any server.
  • Local Control: All cryptographic operations happen locally on your machine.

Disclaimer: This wallet is for educational purposes and should ONLY be used with testnet funds. Do not use with real assets on the Ethereum mainnet.

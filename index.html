Welcome to MiniWallet

A simple, client-side Ethereum wallet for managing your testnet assets.

Create a new wallet, import an existing one, and send transactions with ease.

🔐 Secure & Private

Your private keys never leave your browser. All operations are performed client-side.

🚀 Feature Rich

View transaction history, manage a local address book, and see real-time gas fees.

🛠️ Modern Tech

Built with modern JavaScript (ESM) and Viem for blockchain interactions.
